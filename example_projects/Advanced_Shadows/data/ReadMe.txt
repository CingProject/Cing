Application created with the Cing Application Wizard
-----------------------------------------------------

Steps to start working on your app

1. Right click on your new project and click on "Set as StartUp Project"

2. Right click again and click on "Project Depencencies" -> and mark "_Cing" to set the dependencie with the library

3. Build app (you can do this right clicking again -> "Build"

4. Run app


Note on Data Folder
-------------------

Place in the data folder any data file you want to load from your application (Image, sound files, xml files...etc)

To load them you just need to use the file name (no path required)
